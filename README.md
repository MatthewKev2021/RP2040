# 🚀 Utilidades y Proyectos con el RP2040

Este repositorio está destinado a documentar y almacenar scripts de prueba, librerías y aplicaciones prácticas utilizando el microcontrolador **RP2040** de Raspberry Pi.

## 📌 ¿Para qué sirve el RP2040?
El RP2040 es un microcontrolador de doble núcleo ARM Cortex-M0+ ideal para proyectos donde se requiere control de hardware en tiempo real, bajo consumo y manejo eficiente de periféricos.

## 🛠️ Ideas de Aplicaciones y Pruebas en Simulador

- [ ] **Control de Periféricos:** Pruebas con pantallas OLED (SSD1306) y pantallas LCD mediante protocolos I2C/SPI.
- [ ] **Dispositivos HID USB:** Configuración como teclado o macropad personalizado para accesos rápidos en la compu.
- [ ] **Efectos de Iluminación:** Control de tiras de LEDs NeoPixel utilizando las máquinas de estado PIO.
- [ ] **Lectura de Sensores:** Monitoreo de variables ambientales (temperatura, humedad, distancia).
- [ ] **Automatización Básica:** Control de motores y actuadores para proyectos de robótica o domótica.

## 🔬 Entorno de Simulación
Mientras llega el hardware físico, los diseños y códigos se probarán utilizando el simulador en línea **Wokwi** con soporte para MicroPython.
